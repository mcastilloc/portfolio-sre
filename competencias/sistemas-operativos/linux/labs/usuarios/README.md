# Laboratorio: Gestión de Usuarios en Linux

Este laboratorio demuestra la administración básica de usuarios y grupos en Linux,
incluyendo creación, asignación de grupos y verificación de permisos.

## Objetivos
- Crear usuarios y grupos
- Asignar grupos secundarios
- Verificar información de usuarios
- Aplicar buenas prácticas básicas

## Comandos utilizados
- useradd
- groupadd
- usermod
- passwd
- id

## Escenario
Se requiere crear un usuario de aplicación con permisos controlados
para ejecutar servicios específicos.
