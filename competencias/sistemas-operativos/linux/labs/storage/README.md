# Laboratorio: Gestión de Almacenamiento en Linux

Este laboratorio cubre la inspección y análisis del uso de discos y sistemas de archivos
en Linux.

## Objetivos
- Identificar discos y particiones
- Analizar uso de espacio
- Verificar puntos de montaje

## Comandos utilizados
- lsblk
- df
- du
- mount
